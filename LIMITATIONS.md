# Limitations

This repository asks a narrow evaluation question; it is not a planning tool.

- The world is a generated 3×3 grid with invented labels and prices.
- The included agents are hand-written baselines, not trained policies.
- The reward weights are choices, not validated measures of decision quality.
- No real geography, personal data, LiDAR, listing data, or exhibition media is used.
- A good score here says nothing about an agent's judgment in a real community.

The most useful way to challenge the project is to change the information prices or error penalty and see when the baseline ranking flips.
